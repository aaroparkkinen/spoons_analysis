## Ad Campaign Analysis - Aaro Parkkinen

This repository evaluates the efficiency and profitability of a mobile ad campaign.

results_and_findings.ipynb notebook is the final presentation for the stakeholders to view
containing key results, improvements and future suggestions.

analysis_adperfomrance.ipynb notebook contains supporting material, statistics and plots
relevant for the analysis

cleaned_campaign_data.csv is the pre-processed file used in the analysis
